linkml rendering of RHEA (ALPHA)

  • This repo: chemkg/linkml-rhea
  • Documentation: chemkg.github.io/linkml-rhea
  • Generated artefacts: schema/

This is an ALPHA VERSION of an experimental rendering of the Rhea schema in LinkML

For more on linkml, see:

  • linkml.io/linkml

What are the use cases?

  • providing high level documentation to help form SPARQL queries
  • provide a datamodel that can be used to autogenerate templated queries using sparqlfun
  • Autogenerate
  • SHACL and ShEx for validation of RDF
  • JSON-Schema for validation of JSON
  • Provide a datamodel for submitting data using DataHarmonizer
  • Autogenerate python datamodel for coding
  • Provide mappings to other frameworks, e.g. chemrof
